CLI tool to manage your MongoDB App Services application

Use "appservices [command] --help" for information on a specific command.
To find more documentation on the CLI and commands go to https://www.mongodb.com/docs/atlas/app-services/cli/.
To get started, create project API keys at cloud.mongodb.com/v2/<groupID>/access/apiKeys or call "appservices login"
